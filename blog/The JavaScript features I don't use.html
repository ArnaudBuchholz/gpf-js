The JavaScript features I don't use

Overview

I have been working with JavaScript for several years now but I have a deep background of C/C++ and .NET programming. Most of the time, I try to write 'elegant' code, meaning that I like to make it reliable, efficient and as reusable as possible.

Through this article, I would like to explain why I forbid some keywords and syntaxes from my code. Indeed, based on my experience, they appeared to be more dangerous than helpful. As usual, I am not pretending that I know everything

with
